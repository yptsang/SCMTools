# SCPTools

A unified web platform for supply chain planning that brings together forecasting, inventory segmentation, and data pre-processing tools into a single streamlined workspace.

## Overview

SCPTools is designed for supply chain planners and demand analysts who need to turn raw transactional data into actionable planning decisions. Instead of juggling multiple spreadsheets and ad-hoc scripts, the Hub consolidates four core modules behind one clean interface so that users can move from raw data to insight in just a few clicks.

## Available Modules

(Beyond some data pre-processing modules)
- **Demand Forecasting** — Analyze historical data, detect trends, and generate future demand projections using statistical models.
- **Product Risk Matrix** — Segment SKUs by Temporal Risk and Demand Variability to guide inventory and service decisions.
